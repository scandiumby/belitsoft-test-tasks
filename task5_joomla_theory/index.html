Типы расширений для Joomla! CMS

Плагины

В Joomla это дополнение позволяющие расширить функционал различных элементов CMS (в первую очередь,ядро CMS, компоненты и модули). Плагины привязываются к определённы событиям и могут выполнять определённую дополнительную логику при наступлении этих собыйти. Плагины позволяют модифицировать логику пользовательской и административной частей.

Модули

Модули в Joomla являются небольшими и гибкими расширениями. Модули выводятся за пределами контентной пользовательской часть шаблона.

Предназанчены, в первую очередь, для визуального отоборажения данных. Обычно, можно создавать несколько одинаковых модулей и отдельно их настраивать (в том числе и распологать их в разных позицииях шаблона и на разных страницах). Часто модули тесно взаимодействуют с компонентами и используют их данные.

Компоненты

Компоненты это наиболее функциональные и сложные расширения Joomla. Для вывода компоненты используют контентную часть шаблона.

Обычно состоят из двух частей: пользовательской и административной. В административной части обычно реализуется управление данными и настройка их отображения для пользователей сайта. В пользовательской реализуется отображения данных для пользователей сайта и возможные варианты взаимодействия с этими данными со стороны пользователя.

Шаблоны

Шаблон определяет дизайн и расположение элементов сайта. В шаблне указываются позиции в которые могут выводиться модули и компоненты (контентная часть). В шаблоне возможно переопределять внутренние шаблоны модулей и компонентов для изменения их отображения. Шаблоны могут быть как для пользователской части сайта, так и для административной. Можно привязывать различные шаблоны для различных страниц сайта.

Языки

Система языков Joomla позволяет создавать специальные файлы со списком констант и их значений для разных языков. Эти константы можно использовать во всех расширениях Joomla. C учётом того, что это по сути обычные текстовые файлы - их очень легко редактировать. И вся это система функционирует как дополнение, которые можно перенесить и устанавливать на разные сайты.
